# Sicurezza

Segnala privatamente vulnerabilità e problemi di trattamento dei file tramite una security advisory del repository GitHub.

Easyshop elabora le immagini localmente. La versione base non carica fotografie, metadati o prompt su servizi esterni. Eventuali integrazioni cloud future dovranno essere opzionali, dichiarate e disattivate per impostazione predefinita.

Non allegare immagini private o file contenenti dati sensibili a issue pubbliche.
