# Changelog

## 0.1.0 Alpha — 2026-07-21

First public alpha of Easyshop.

- Native universal macOS app for Apple Silicon and Intel.
- Raster, text and adjustment layers with editable masks.
- Rectangle, ellipse, lasso, brush and on-device Apple Vision subject selection.
- Honest ML provenance: local algorithmic fallbacks are never labelled AI.
- Light, colour, HSL, temperature, tint, curves, detail and local adjustments.
- Image resize, canvas resize, crop, DPI and multiple resampling methods.
- Open `.easyshop` project format and ImageIO-based import/export, plus compatible PSD export.
- Movable, collapsible Liquid Glass tool and inspector islands.
- Transactional size changes, bounded imports and automatic crash recovery.
- No account, telemetry or mandatory cloud service.

This build is ad-hoc signed and not Apple-notarized. On first launch, Control-click Easyshop and choose **Open**.
